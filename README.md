> [Click me download source code / Try it online](https://www.devquizdone.online/detail/7a79894c543a4cf5a44cb318cbeb8d61/ghb20250919) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Feature Description
The system is divided into (roles): Regular users, Teachers, Administrators.

**Regular User Features:**
1. Login
2. Registration
3. Change Password
4. Personal Information
5. Class Information
6. Leave Application Form
7. Submit Leave Application
8. Student Attendance
9. Absence Records


**Teacher Features:**
1. Login
2. Registration
3. Change Password
4. Personal Center
5. Student Information
6. Class Information
7. Leave Application Form Management
8. Submit Leave Application Form Management
9. Student Attendance Management
10. Absence Records Management



**Administrator Features:**
1. Login
2. Personal Center
3. Student Management
4. Teacher Management
5. Class Information Management
6. Leave Application Form Management
7. Submit Leave Application Management
8. Student Attendance Management
9. Absence Records Management
